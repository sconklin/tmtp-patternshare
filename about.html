About Us

This site is a collaboration between the Tau Meta Tau Physica and The Pattern Share project.

The Tau Meta Tau Physica project was founded by Susan Spencer to meet a need for digital pattern making. In the last two years, the project has been implemented as a batch-mode application written in Python, and is currently being converted to an HTML5 web application.

The The Pattern Share project is a masters project by designer Justin Couch, who attends the Rhode Island School of Design.

Major code contributions to the TMTP project were made by Steve Conklin, an open source software developer.

Web site development and styling by Molly Richardson, founder of MR 21 Solutions.
